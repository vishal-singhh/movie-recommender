# Movie Recommender System — TMDB Dataset

A content-based movie recommender using cosine similarity on genres, keywords, cast, crew, and overview.

## Files
```
app.py               ← Streamlit app
build_model.py       ← Run once to generate model files
requirements.txt     ← Python dependencies
model/
  movie_list.pkl     ← Pre-built movie DataFrame (generated by build_model.py)
  similarity.pkl     ← Pre-built cosine similarity matrix
tmdb_5000_movies.csv
tmdb_5000_credits.csv
```

## Local Run

```bash
pip install -r requirements.txt
python build_model.py   # only needed once, or if model/ folder is missing
streamlit run app.py
```

## Deploy to Streamlit Community Cloud

1. Push this entire folder (including `model/`) to a **GitHub repository**.
2. Go to [share.streamlit.io](https://share.streamlit.io) → **New app**.
3. Point it at your repo, set **Main file path** to `app.py`.
4. Click **Deploy** — Streamlit Cloud installs `requirements.txt` automatically.

> ⚠️ The `model/` folder (pkl files) must be committed to the repo since
> Streamlit Cloud does not run `build_model.py` automatically.  
> The two pkl files total ~180 MB — make sure Git LFS is enabled if your
> repo host requires it for files that large.

## How It Works

- Movie metadata (genres, keywords, top-3 cast, director, overview) is
  combined into a "tags" string per movie.
- A `CountVectorizer` (5 000 features, English stop-words removed) converts
  the tags to a bag-of-words matrix.
- Cosine similarity between all movie pairs is pre-computed and saved.
- At query time the app fetches the 5 most similar movies and retrieves
  their posters from the TMDB API.
